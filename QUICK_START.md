# ⚡ دليل البدء السريع

## 🎯 ما تحتاج إلى فعله الآن

### 1️⃣ نسخ الملفات الجديدة (5 دقائق)

```bash
# انسخ هذه الملفات إلى جذر موقعك:
✅ styles.css
✅ app.js
✅ sw.js
✅ manifest.json
✅ index.html (استبدل القديم)
✅ dashboard.html (استبدل القديم)
```

### 2️⃣ تحديث رأس HTML (2 دقيقة)

أضف هذا في `<head>` لكلا الملفين:

```html
<!-- الأنماط الجديدة -->
<link rel="stylesheet" href="styles.css">

<!-- دعم PWA -->
<link rel="manifest" href="manifest.json">
<meta name="theme-color" content="#0b6b42">

<!-- قبل </body> -->
<script src="app.js" defer></script>
```

### 3️⃣ اختبر الموقع (3 دقائق)

```
1. افتح https://your-website.com
2. اضغط F12 لفتح وحدة التحكم
3. ابحث عن: ✅ تم تهيئة التطبيق بنجاح
4. جرّب الوضع الداكن (اضغط على زر الثيم)
```

## 🐛 إذا حدثت مشكلة

### الأنماط لا تظهر؟
```
✓ تحقق من أن styles.css في نفس المجلد
✓ امسح ذاكرة التخزين المؤقت (Ctrl+Shift+Delete)
✓ أعد تحميل الصفحة (F5)
```

### JavaScript لا يعمل؟
```
✓ افتح وحدة التحكم (F12)
✓ ابحث عن رسائل الخطأ الحمراء
✓ تأكد من أن app.js موجود
```

### Supabase لا يتصل؟
```
✓ تحقق من مفاتيح Supabase
✓ تأكد من أن المتصفح يسمح بـ CORS
✓ اختبر الاتصال في وحدة التحكم
```
